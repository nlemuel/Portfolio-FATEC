## Projeto Integrador: 2023-1

## Projeto 5º Semestre: ***PowerTech***.

### Introdução ao projeto

A IACIT é uma empresa brasileira que possui capacitação tecnológica para o desenvolvimento de diversos produtos.
Atualmente, os clientes enviam ROs (Registros de Ocorrência) por vários meios de comunicação: aplicativos, e-mail, telefone, etc.
O objetivo do projeto é facilitar essa comunicação desenvolvendo um aplicativo móvel para tal função.

### Empresa parceira

IACIT

***

### Visão do Projeto

O Objetivo é criar uma aplicação híbrida/responsiva, que funcione como Serviço de Atendimento ao Cliente (SAC), onde ele poderá criar ROs, verificar as solicitações criadas, entrar em contato via chat com algum administrador (programador da equipe). Assim, os administradores podem receber as solicitações, atualizar seus status e verificar as pendências de forma centralizada.

### Solução

Utilizamos React Native para o desenvolvimento mobile focando na experiência do usuário e sua comunicação com os administradores do sistema via chat. A atualização dos registros de ocorrência é feita em tempo real para agilizar o atendimento. Também criamos relatórios para facilitar a visualização e sumarização dos registros de ocorrência.

### Lista de Requisitos 

Para a validação das entregas, utilizamos os seguintes critérios 

### Requisitos Funcionais:
- Área de Cliente e área de Administrador, com controles de acesso (autenticação/autorização) 
- Opções de criar e acompanhar ROs 
- Comunicação com os administradores do sistema via chat 
- Atualização do Status das ROs em tempo real
- Envio de e-mail com notificações para o cliente e administradores 
- Relatórios/gráficos de sumarização das ROs

### Requisistos Não Funcionais
- Documentação com mapeamento das regras de negócio do cliente.
- JavaScript
- TypeScript
- React e ReactNative
- Python
